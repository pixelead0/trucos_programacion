## [1.0.0] - 2024-01-XX

### Agregado
- Curso completo de Python desde fundamentos hasta conceptos avanzados
- 9 módulos de aprendizaje progresivo
- Proyectos prácticos con versiones evolutivas
- Documentación en Docusaurus
- GitHub Actions para despliegue automático
- Lecciones sobre:
  - Fundamentos de Python
  - Estructuras de datos
  - Funciones y módulos
  - Programación orientada a objetos
  - Manejo de errores y buenas prácticas
  - Manejo de archivos y formatos
  - Conceptos avanzados (decoradores, context managers, generadores)
  - Herramientas profesionales (virtual envs, packaging, testing, security)
  - Ciencia de datos básica (Pandas)
